# 01 – Dashboard (HTML + CSS + JS)

This folder contains the **EventHub360 EMS Dashboard**, split into clean, separate files.

## Files

| File | Purpose |
|------|---------|
| `index.html` | HTML structure — auth screen, sidebar, page shell |
| `style.css`  | All CSS — variables, layout, components, responsive |
| `app.js`     | All JavaScript — data, auth, routing, page builders, charts |
| `eventhub360-dashboard.html` | Self-contained single-file version (same app, all-in-one) |

## How to open

**Option A – Split files (recommended for development):**
Open `index.html` in any browser. `style.css` and `app.js` are loaded automatically.

**Option B – Single file:**
Open `eventhub360-dashboard.html` directly — no other files needed.

> ⚠️ Both options require Chart.js from CDN, so an internet connection is needed on first load.

## Quick login
Hit the **🎓 Demo Login (Admin)** button — no account needed.
